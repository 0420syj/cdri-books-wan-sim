## ⚙️ 실행 방법 및 환경 설정

### ▶️ 실행 방법
```bash
pnpm i
pnpm dev
```

### 🛠️ 환경 설정
1. `.env` 파일을 프로젝트 루트 디렉토리에 생성합니다.
2. 이메일로 전달받은 환경 변수를 아래와 같이 추가해주세요.
```env
NEXT_PUBLIC_KAKAO_REST_API_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
